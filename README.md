# XToll: eXtreme-weather Toll Tracker in the contiguous United States

XToll is an interactive dashboard which visualizes the county-level rankings for wildfire smoke-, heat-, cold-, drought-, and flood-related and total extreme weather event-related mortality. XToll also depicts the state-specific "Dollars Lost to Extreme Weather-Related Mortality" for the state in which the county is located. State-level monetary values are calculated to ensure individual health records remain anonymous.
